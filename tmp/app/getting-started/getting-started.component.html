Getting started

Welcome to Slamby. Congratulation, you made the very first step to work with our awesome products. You are almost done, but before getting started working with your Slamby product, here are a few useful information.

API install

You can use your Slamby API as a SaaS, or you can download, and host is by yourself. When you want to host it by yourself, the first thing you need to manage is the installation. To install Slamby API, please check the product readme and the installation guide.

Product documentation: https://github.com/slamby/slamby-api

Using Docker Compose, here are the installation steps:

  1. Install Docker on your machine: Official Docker installation guide
  2. Install Docker Compose on your machine (minimum 1.9.0 required): Official Docker Compose installation guide
  3. Download our Docker Compose file
  4. ...

API activation

Great, you have an installed Slamby API. Now your first step is the activation process.
During the setup process you need to:

  • Request a Slamby License,
  • Copy your Slamby API License,
  • and set the secret (password) for your Slamby API.

License
For open-source projects, you can use an `open-source license` for free, or for commercial purpose you can use a commercial license.

For open-source license, your can request one from the setup wizard directly by typing your e-mail address and click the `Request a License` button. Our license service will generate an open-sourced license for you and will send it via email to the given email address.

When your product or company is not open-sourced you cannot use Slamby API with open-source license; you need to purchase a `commercial license`. The pricing is based on the processor core number.

For pricing details, please check our pricing https://www.slamby.com/pricing

For purchasing a commercial license contact our sales@slamby.com

Support

Slamby support mailing list for basic support: https://groups.google.com/forum/#!forum/slamby

Each Slamby license includes a `basic support` with absolutely no SLA. This means you can access our support via our online support mailing list and you can ask for help from our community. In this case, the reaction time is not guaranteed, and there is no direct access to our technical experts.

When you need a `professional support`, please purchase one. Each professional support contains short-time SLAs and response time. Each professional support gives you direct access to our technical experts.

For more details, please check our available support options here: https://www.slamby.com/support

And contact our sales: sales@slamby.com
